# 📖 Overview

Understanding where restaurants are located and how they perform can reveal valuable business insights.

This project analyzes restaurant data using **geospatial analytics** and **exploratory data analysis (EDA)** to identify trends across cities and localities. It combines statistical analysis with interactive visualizations to examine restaurant distribution, ratings, pricing, and cuisine diversity.

The project demonstrates practical applications of **location intelligence** and **data-driven decision making**.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/location-based-restaurant-analysis.git
```

Navigate to the project

```bash
cd location-based-restaurant-analysis
```

Install dependencies

```bash
pip install pandas numpy matplotlib plotly folium
```

Run the analysis

```bash
python locality_insights.py
```
